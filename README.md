# 🃏 HobbyProfit Tracker

A sleek, premium, single-page inventory ledger and financial tracking application designed explicitly for modern sports card dealers, flippers, and hobbyists. Optimized for rapid, one-handed mobile use on the card show floor.

## ✨ Key Features

* **Dynamic Inventory Ledger:** Effortlessly log card details, acquisition costs, and track statuses (Raw, At Grading, Listed for Sale, Sold).
* **Smart Financial Dashboard:** Real-time metrics tracking total capital invested, realized net profit, ROI, and active tax liabilities.
* **COGS Aging Monitor:** A protective warning system that flags stagnant capital and aging inventory.
* **Conditional "Listed Price" Assistant:** Tracks live marketplace asking prices against your absolute bottom-dollar profit thresholds.
* **Social Display Card Generator:** Instantly transform card data into gorgeous, high-contrast, screenshot-ready digital graphics for Instagram Stories, Twitter/X, or Discord.
* **High-Tolerance CSV Importer:** Smart data-ingestion mapping that handles irregular spreadsheet column headers without breaking calculations.
* **Full Mobile Optimization:** Built from the ground up for touchscreen viewports with native mobile inputs and overlay scroll-locking.

## 🛠️ Technology Stack

* HTML5 / Semantic Markup
* Tailwind CSS (via CDN) for modern, utility-first styling
* Vanilla JavaScript (ES6+) for rapid, client-side state execution
* Browser `localStorage` API for instant, private data persistence

## 🚀 Quick Start

1. Download or clone this repository.
2. Double-click the `index.html` file to open it in any modern web browser.
3. *Mobile Use:* Host via GitHub Pages to save as a progressive web shortcut on your smartphone's home screen.

## 🔒 Privacy & Licensing

All data is processed and stored locally within your browser's private storage ecosystem. No financial information is ever transmitted to external servers. This project is open-source and available under the MIT License.
